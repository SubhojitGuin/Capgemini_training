# Steps to Convert Java Project into JAR file:-
-----------------------------------------------

- Click on the Project right click on it select export as an option
- Search as JAR In the Wizard
- Select the jar file option under the JAVA folder
- Click on browse to show the path where the jar file should be created in the system
- Name the jar file with the .jar as an extension
- Click on finish

## Java Build Path
- It is the process of showing the path of the jar into the Java Project
- By adding a JAR file to the Java Build Path, we can use the classes and methods present
in that JAR file inside our project.

# Steps to Java Build Path
--------------------------------
- Click on the Project right click on it and select Build Path move the cursor to the right
and select Configure build path
- Click on libraries and click on Add External Jars
- Select the required Jar should be imported into the Project
- Click on finish
